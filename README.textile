h1. Grape

h3. Graph Editor

p. Grape is a graph editor intented to ease the graph external representation
and algorithms simulation, be it for regular learning or for applying A.I..

p. Written in Python it is released under the GNU General Public License and
uses the MVC pattern, based on the Ruby on Rails web application framework it
ports the great concepts of it to a standard, general desktop application.
